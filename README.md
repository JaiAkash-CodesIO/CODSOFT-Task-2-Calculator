Simple Calculator
This is a basic calculator application built using Python and Tkinter. It supports the four fundamental arithmetic operations: addition, subtraction, multiplication, and division.
The GUI is designed to be intuitive and easy to use, making it a perfect project for learning and practicing Pythonâ€™s Tkinter library.
Features
Addition (+): Add two numbers.
Subtraction (-): Subtract one number from another.
Multiplication (*): Multiply two numbers.
Division (/): Divide one number by another, with division by zero handled gracefully.
Clear (C): Reset the input field.
Equals (=): Evaluate the current expression and display the result.
Prerequisites
Ensure that Python is installed on your system. 
Usage
Click the number buttons (0-9) to input numbers.
Choose an operation (+, -, *, /) to perform on the numbers.
Click the = button to evaluate the expression.
Use the C button to clear the input field and reset the calculation.
